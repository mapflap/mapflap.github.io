LLMs for location detection of terroristic events

This technical report discusses the potential use of Large Language Models for location detection with a focus on the JRC Terrorism and Extremism Database. The report highlights the current inaccuracies in the database's location detection algorithm, which uses adhoc created embeddings, struggles with contextual issues, and has difficulty with translated location names.
